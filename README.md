# Apple-Style Calculator

A beautiful web calculator inspired by the Apple Calculator design. Built with vanilla HTML, CSS, and JavaScript.

## Features

- Clean Apple-inspired dark theme
- Basic operations: +, −, ×, ÷
- Percent, toggle sign (±), clear
- Responsive design
- Smooth button interactions

## Local Development

Open `index.html` in your browser or use a local server:

```bash
npx serve .
```

## Deploy to Vercel

1. Push this repo to GitHub (see below)
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click "Add New Project" → Import your GitHub repo
4. Deploy (no build settings needed—it's a static site)

Or use the Vercel CLI:

```bash
npx vercel
```
